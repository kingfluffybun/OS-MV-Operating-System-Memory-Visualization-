What is OS-MV?

OS-MV is a desktop-based memory management visualizer designed to help students and educators understand how operating systems allocate memory to processes. Memory management is a core concept in operating systems — how memory gets allocated directly affects system performance, efficiency, and stability. It supports seven allocation algorithms — First Fit, Next Fit, Best Fit, and Worst Fit under contiguous allocation (both fixed and dynamic partitioning), as well as Paging, Segmentation, and Segmentation with Paging under non-contiguous allocation.

How to Get Started?

OS-MV is simple to use! Just pick a mode, select an algorithm, add your processes and memory blocks, and hit Play. Use Next to step through each allocation one at a time, or adjust the speed slider to control how fast the simulation runs. Reset anytime to start fresh with new inputs.

1

Choose a mode — Single Mode for one algorithm, Comparison Mode to run all at once and compare results.

2

Pick an algorithm — Select from Contiguous (Fixed or Dynamic) or Non-Contiguous options.

3

Add processes — Enter a size in KB and click , or use the button for a random size.

4

Configure memory — Add memory blocks or set memory/page size depending on the algorithm.

5

Run — Hit Play to run automatically or Next to step through one at a time.
